Drip form 参数

了解Drip form 支持传递哪些参数

📄️ unitedSchema

表单协议

📄️ uiComponents

导入主题、自定义主题

📄️ formData

设置表单初始化数据

📄️ customComponents

导入自定义组件

📄️ ref

获取表单数据、api、配置联动

📄️ globalData

全局数据

📄️ control

监听内部数据变化

📄️ parse

表单渲染前修改数据

📄️ transform

表单提交前修改数据

📄️ onValidate

设置自定义校验或异步校验

📄️ onQuery

异步获取表单数据

📄️ onEvent

事件回调

📄️ onMount

组件didMount函数

📄️ onSubmit

表单提交按钮回调

📄️ onCancel

表单取消按钮回调

📄️ plugins

加载自定义校验插件

📄️ options

dripform自定义配置

📄️ fetchApi

fetchApi 用来实现 Drip form 内部接口请求。

📄️ customFunc

函数占位符

📄️ containerHoc

容器高阶函数

📄️ ajvOptions

自定义ajv option
