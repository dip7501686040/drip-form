介绍

@jdfed/form-generator是 drip-form 的快速生成界面。

功能介绍​

  • 快速搭建 drip-form 表单
  • 生成表单 JSON 数据
  • 导入 JSON 数据，预览表单
  • 可扩展（头部配置、组件配置、viewport 配置、属性配置、校验配置）

区域介绍​

generator 生成器分四个区域：

  • header

    header 区可以导入 json、编辑 json、预览表单、导出 json

  • 组件区

    组件区包含 generator 支持拖拽的表单控件。

    • 表单组件

      常见表单控件

    • 辅助组件

      非常见表单控件，用于展示文字等

    • 容器组件

      根据数据类型选择对应容器

      • 数组容器

        支持自增、元祖

      • 对象容器

        支持折叠、分类展示表单

    • 业务组件

      业务自定义组件，可根据业务自定义

  • 可视区

    实时查看组件区拖拽

  • 编辑区

    编辑区可以编辑表单样式、表单校验

    • 属性配置

      分为全局配置和表单配置

      点击可视区空白区域为全局配置；点击选中可视区表单为表单配置

    • 校验配置

      根据当前表单数据类型，配置数据校验
