Radio 单选框

outline​

solid​

配置​

该字段配置在schema中：

schema
{
    "type": "string",
    "title": "单选",
    "ui": {
        "type": "radio",
        "buttonStyle": "outline",
        "options": [
            {
            "label": "是",
            "value": "0"
            },
            {
            "label": "否",
            "value": "1"
            }
        ],
        "disabled": false
    },
    "fieldKey": "radio_qYjFa-"
}
提示

组件具体配置依赖于所使用的UI库，DripForm仅作示例，具体可参考UI库对应文档。

API​

参数说明类型是否必填默认值
filedKey表单数据存放字段（必须唯一）string必填-
options可选数据源options[ ]必填-
buttonStyleRadioButton 的样式，目前仅支持两种风格outline、solid非必填outline
disabled值为 true 时，单选框为禁用状态boolean非必填false
onChange选项变化时的回调函数function(e:Event)非必填-

Options​

options {
    value: string | number;
    label?: React.ReactNode;
    disabled?: boolean;
}
