errorMessage 错误提示

drip-form 是实时校验的。不过可以通过配置dataSchema来控制是否展示错误信息

dataSchema的validateTime字段为change时，将会展示表单错误信息在表单下方。

不配置validateTime字段时，默认不展示错误信息

使用场景​

  1. 实时在表单下方展示错误信息

    dataSchema.json
    {
        "$schema": "http://json-schema.org/draft-07/schema#",
        "type": "object",
        "validateTime": "change",
        "properties": {
            "name": {
                "title": "名字",
                "type": "string",
                "transform": ["trim"]
            },
            "name1": {
                "title": "名字1",
                "type": "string",
                "transform": ["trim", "toLowerCase"]
            }
        },
        "required": ["name", "name1"],
        "errorMessage": {
            "required": {
                "name1": "必填",
                "name": "必填"
            }
        }
    }

  2. 表单提交才展示错误信息

    1. 首先设置 dataSchem 的 validateTime 字段为submit

      dataSchema.json
      {
          "$schema": "http://json-schema.org/draft-07/schema#",
          "type": "object",
          "validateTime": "submit",
          "properties": {
              "name": {
                  "title": "名字",
                  "type": "string",
                  "transform": ["trim"]
              }
          },
          "required": ["name"],
          "errorMessage": {
              "required": {
                  "name": "必填"
              }
          }
      }

    2. 点击提交按钮，使用ref.current.dispatch设置 validateTime 为 change

      const Form = () => {
          const ref = useRef(null)
          const onSubmit = () => {
              const { dispatch } = ref.current
              // 点击的时候才展示错误信息
              dispatch({
                  type: 'setValidate',
                  action: {
                      set: {
                          validateTime: 'change',
                      },
                  },
              })
          }

          return (
              <>
                  <DripForm
                      ref={ref}
                      dataSchema={dataSchemaMult}
                      uiSchema={uiSchemaMult}
                      uiComponents={{ 'drip-design': DripDesign }}
                  />
                  <Button
                      type="primary"
                      className="button--submit"
                      onClick={() => {
                          onSubmit()
                      }}
                  >
                      提交表单
                  </Button>
              </>
          )
      }
    提示

    drip-form 总是会将错误记录下来，最后可以通过ref.current.checking和ref.current.errors获取校验状态以及错误信息。详情查看
